Displays Skemboo feeds as wordpress widgets

== Description ==

This plugins allows display of Skemboo's feeds into widgets.
Actually in beta, many changes can occure during the next releases.

== Installation ==

1. Upload skemboo-widget directory to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in Wordpress
3. Go to 'Widgets' menu in Wordpress and place in a widget placeholder
